# llm-precommiter

## 概要
`llm-precommiter`は、Gitのpre-commitフックを使用して、コミット前にAIエージェントがコードをチェックするためのモジュールです。このツールは、コードの品質を向上させ、潜在的なバグを早期に発見することを目的としています。
